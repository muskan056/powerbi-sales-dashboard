# Sales Performance Dashboard - Power BI

## Project Overview
This Power BI dashboard provides insights into sales performance, profit trends, customer behavior, and regional analysis.

## Features
- KPI Cards
- Sales & Profit Analysis
- Interactive Slicers
- Month-over-Month Growth
- Regional Performance Insights
- Dynamic Dashboard Design

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel Dataset

## Files Included
- powerbi-sales-dashboard.pbix
- Power-BI-Dataset.xlsx
